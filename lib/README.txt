Download the JUnit 5 platform console standalone jar from:
https://search.maven.org/remotecontent?filepath=org/junit/platform/junit-platform-console-standalone/1.10.0/junit-platform-console-standalone-1.10.0.jar

Place the downloaded jar in this lib/ directory.

Example download command:
curl -L -o junit-platform-console-standalone-1.10.0.jar \
  https://search.maven.org/remotecontent?filepath=org/junit/platform/junit-platform-console-standalone/1.10.0/junit-platform-console-standalone-1.10.0.jar
